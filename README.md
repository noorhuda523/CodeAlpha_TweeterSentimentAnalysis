# CodeAlpha_TweeterSentimentAnalysis
Twitter Sentiment Analysis using NLTK VADER on 50K tweets from Sentiment140 dataset. Classifies public opinion as Positive, Negative &amp; Neutral. Includes text cleaning, compound scoring, 6 visualizations, and WordCloud. Built with Python, Pandas, Matplotlib, Seaborn &amp; WordCloud. | CodeAlpha Internship
